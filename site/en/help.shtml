How to create a note?

To create a new note, move the mouse to the input field on the top which is marked as: " write down you note title and press 'Enter'".

Please think of the title of this note, like: what's this note about? Or just the content of the note directly if it's short enough. Then press key "Enter", Midinote will create a new note for you.

How to input tags

There are several different way to input tags.

First, when you are writing note, press key # to begin to input a tag.

Keep typing the name of the tag, you can not include any space char in tag's name, and press space to finish inputting tag.

Or, when you pressed # and has been in the mode of inputting tag, Midinote would give you some suggestion if any.

Using up and down to change the option, press enter to pick the tag you want.

Another way to input tags is by using the tag recommender bar. On top of the app, there is a rectangle area called the tag recommender bar. As you typing note content, Midinote would analyze the content, and give you some recommended tag in the bar.

Just click the recommended tag directly to input the tag into your notes.

And, you can also input tag in this way:

Select the text that you want it to become a tag. Then, click the # icon in the toolbar. The text selected would become a tag.

How to develop my mind map structure in Midinote?

In Midinote, you can develop your own mind map structure by defining tags. Say you have a tag 'A' and 'B', if you using 'A' to define 'B', then you created a structure that B belongs to A, or you can say, B is a sub-tag of A. Presented in the mind map, the B would be a branch node linked to A.

First, let's show you how to define a tag. To define a tag, move mouse to the tag which you want to define. There would be a tooltip to be shown.

Click the 'gear' icon to open the tag setting dialog.

Find the section: "definition".Input something about: What is this tag ? Or, what is this tag about? And you can input tags just like you do it in notes.

When you defined the relationship between tags, you will find the mind map would represent it.

And you will find, all those notes belong to the children tags, would belong to their parents too.

What's article?

In Midinote, when you login, what's presenting to you is about an 'article'.

As shown below, there are some elements in an article, let's discover it one by one.

Article Title

The title of current article, actually, the must be a tag name. In the world of Midinote, every tag is an article. You can click any tag to go into the article of this tag.

Article Navigator

The navigator indicate the position/path to current article. Like what illuminated in the figure, the current article/tag is 'B', it's parent is 'A', and A's parent is the unique global tag: "Merry", the so-called unique global tag, is the first tag created when you begin to use Midinote, all other tags in Midinote belong to it.

The navigator also record the history of your visit between articles/tags. Using or to go back or forward in your visit history.

Article Description

Actually, they are the definition of current tag.

Article Content

All the notes belong to this article/tag.

In Midinote, if you attach a tag to a note, then, this note belongs to the tag. It would appear in the article of this tag. Also, this note would appear in all the ancestor of this tag.

Article Graph

Show this article in graph. There are two different types of graph:

Mind Map

Bubble

Click icon or to switch between these two graph.
